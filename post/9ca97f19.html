Linux基础知识总结

发表于2024-05-22|更新于2025-09-02|操作系统
|总字数:1.7k|阅读时长:6分钟|浏览量:

常见操作系统：Windows，IOS，Android，MacOS，Linux，Unix

Linux主要发行版（Linux是一个内核，以此开发的）：Ubuntu（乌班图）、RedHat（红帽）、CentOS、Debain（蝶变）、Fedoras、SuSE、OpenSUSE

GNU Linux

![[/Untitled 87.png|Untitled 87.png]]

![[/Untitled 1 7.png|Untitled 1 7.png]]

  1. /bin:
    • 存放系统启动和修复过程中所需的基本可执行文件，包括一些最基本的命令。
  2. /boot:
    • 包含启动系统所需的内核和引导加载程序文件。
  3. /etc:
    • 存放系统配置文件的目录，包括网络配置、软件包配置等。
  4. /home:
    • 用户的主目录通常位于这里，每个用户都有一个单独的子目录。
  5. /lib 和 /lib64:
    • 包含系统和应用程序所需的共享库文件。
  6. /media 和 /mnt:
    • 用于挂载可移动媒体设备（如USB驱动器）和其他文件系统的目录。
  7. /opt:
    • 通常用于存放额外安装软件的目录，与系统软件包无关。
  8. /proc:
    • 是一个虚拟文件系统，提供有关系统内核状态的信息。
  9. /root:
    • 超级用户（root）的主目录。
  10. /sbin:
    • 存放系统管理员使用的系统管理命令（例如，系统修复、网络设置等）。
  11. /srv:
    • 通常用于存放服务（如Web服务器）的数据目录。
  12. /tmp:
    • 临时目录，系统和用户可以在此处创建临时文件。
  13. /usr:
    • 包含用户安装的软件包、程序和文件。子目录如下：
      • /usr/bin: 用户可执行文件。
      • /usr/lib: 用户库文件。
      • /usr/sbin: 系统管理员使用的系统管理命令。
      • /usr/share: 共享数据，如文档、图标等。
      • /usr/local:另一个给主机额外安装软件所安装的目录，一般通过编译源码方式安装的程序
  14. /var:
    • 存放经常变化的文件，如日志文件、数据库文件等。子目录如下：
      • /var/log: 存放系统日志文件。
      • /var/spool: 存放一些服务的工作目录，如邮件服务的邮件队列。

vi 文本编译器

**ls**显示当前工作目录中的内容

  • **ls -l**: 以长格式列出文件和目录的详细信息，包括文件权限、所有者、文件大小等。
  • **ls -a**: 显示所有文件，包括以点（**.**）开头的隐藏文件。
  • **ls -lh**: 以长格式显示文件大小，并以人类可读的格式（例如 KB、MB）显示文件大小。 **-R**: 递归地列出所有子目录的内容。
  • **ls -t**: 按修改时间排序，最新的文件或目录在前面。

pwd查看当前目录

cd切换目录

~ 代表当前用户的主目录

.代表当前目录
. .代表上一级目录
cd ~切换到主目录
cd ~/myself 切换到主目录下的myself目录

cd …/snap切换到上级目录、再到snap子目录

ls ~

目录、文件相关操作

mkdir创建目录

mkdir -p abc/123/test 创建多层次目录

rmdir abc删除空目录，非空则失败

rm -rf abc 强制删除abc目录，和子项一并删除

cp 复制文件或者目录cp -rf example example2
mv 移动文件或目录(重命名)move hello helloworld

归档

tar -cvf xxx.tar file1 file2 将多个文件或目录打包成一个单一的文件

还原档案包 tar -xvf xxx.tar -C outdir -C表示切换一个目录_，_不写默认放到当前目录里

归档并压缩

tar -czvf xxx.tar.gz xxx

第一个xxx是归档包的名字，第二个xxx是里面的内容文件

解压缩

tar -xzvf xxx.tar.gz

tar -xzvf xxx.tar.gz -C outdir

软链接→window下桌面的快捷方式

ln -s xxx xxx’

xxx是软链接里面的内容，xxx’是快捷方式的名字

删除软链接对原文件无影响，反之不然

ls -l 可以展示文件信息

sudo表示以管理员身份执行 sycyyds513

设置密码 sudo passwd lr

删除用户 sudo userdel lr

删除旧用户遗留的目录 sudo rm -rf /home/lr/

超级用户 不用加sudo

首次使用，设置密码 sudo passwd root

切换到root用户 su root
